# `Social-Media`

- __Social-Media is social media application for sharing experience, favourite photos memories, emotions and locations, invite/follow friends, private inboxing through the application.__

## `Built With`

- PHP
- MYSQL
- HTML5
- CSS3
- JAVASCRIPT
